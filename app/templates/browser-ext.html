What is this extension?

This browser extension lets you download a .pose file with the preview image, author, and tags off XIV Mod Archive without needing to merge them manually to this site. It's designed to save you a few clicks!

If you're on xivmodarchive.com and go to a pose page, it adds a button to the mod page.

Pose pages without a .pose file will say "Pose file not detected"

This also makes it easier to search for saved poses using tags in posing tools that support it!

How it works (The technical-ish bit)

The extension uses a script (content.js) that hangs out on xivmodarchive.com. When you're looking at a mod page, it sticks a "Download Pose w/ Image" button right there in the download section.

Here's what it's doing behind the scenes:

  • It finds the download link and checks if it's actually a .pose file.
  • It grabs the preview image, tags, and the author's name from the mod page.
  • It resizes the image down to 720p (and even tries to crop out those annoying black bars) and stuffs it all into the .pose.
  • Finally, it hands you the updated file directly. No need to visit this site at all if you're just grabbing stuff from the XIV Mod Archive.

It's basically a "one-click" version of what this site does for those who are downloading poses, specifically tailored for xivmodarchive (For now). It's still might need some work, but it saves a lot of time.
